# NYT Reply Bot

A mastodon bot that looks for NYT Games social shares, Cooking links, and articles to remind folks that we're on strike.

## Getting started

### What you'll need

- Go (latest version)
- [Task](https://taskfile.dev/installation/). See [Taskfile](./Taskfile.yml)

### Local development

```shell
task run
```

```shell
task build
```
